# CS-370-Portfolio
CS 370 Module Eight Journal
Briefly explain the work that you did on this project
In this project, I built a pirate intelligent agent for the Treasure Hunt Game. The goal was to train the agent to move through an 8x8 maze and reach the treasure successfully. Instead of using a fixed pathfinding method, I used deep reinforcement learning so the agent could learn through trial and error. This project helped me understand how reinforcement learning and neural networks can work together to solve decision-making problems.

What code were you given? What code did you create yourself?
I was given starter code for the project, including the maze environment, helper functions, and parts of the model structure. I created the main training logic for the pirate intelligent agent. My work included implementing the deep Q-learning process, balancing exploration and exploitation, storing experience in memory, training the model over many episodes, and improving the agent’s performance over time.

What do computer scientists do and why does it matter?
Computer scientists solve problems by designing algorithms, building software, and creating systems that help people and organizations. Their work matters because technology is used in almost every part of daily life, including business, health care, education, communication, and security. In this course, I learned that computer science is not only about writing code but also about choosing the right approach to solve a problem effectively.

How do I approach a problem as a computer scientist?
I approach a problem by first understanding the goal and then breaking it into smaller parts. In this project, I had to understand the environment, the agent’s possible actions, the reward system, and how the learning process worked. After that, I focused on testing, training, and improving the model step by step. This showed me that problem solving in computer science requires patience, logical thinking, and continuous improvement.

What are my ethical responsibilities to the end user and the organization?
My ethical responsibility is to build systems that are reliable, safe, and fair. I should think about how the end user may be affected by the system and avoid creating solutions that are careless or harmful. I also have a responsibility to the organization to write honest, maintainable, and secure code. In artificial intelligence projects, ethics is especially important because the system’s decisions can affect real outcomes.
